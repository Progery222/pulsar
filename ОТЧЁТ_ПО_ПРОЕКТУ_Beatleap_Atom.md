# Отчёт по проекту «Beatleap Atom» (Pulsar)

**Куратор проекта:** Алексеев Денис, начальник сектора автоматизации
**Тип:** десктопное приложение для Windows (внутренний инструмент автоматизации подготовки видеоконтента)
**Период активной разработки:** 23 июня — 13 июля 2026 (≈3 недели)
**Объём работ:** 348 коммитов; ~26 000 строк основного кода + интегрированный графический редактор (617 файлов)

---

## 1. Резюме для руководства

За три недели силами команды под кураторством сектора автоматизации разработан и доведён
до релизной сборки **комплексный десктопный продукт для массового производства коротких
видео под соцсети**. Продукт закрывает полный цикл — от скачивания исходников до выпуска
десятков уникальных роликов — и заменяет связку из нескольких платных онлайн-сервисов
одним офлайн-приложением со встроенными движками (FFmpeg, Python-нейромодули).

Ключевой эффект автоматизации: операции, которые раньше делались вручную в сторонних
редакторах (монтаж под музыку, уникализация под антидубли, чистка чужих титров, озвучка,
дубляж), сведены в единый управляемый конвейер с очередью задач, историей и пакетной
обработкой.

---

## 2. Состав продукта (что реализовано)

Приложение построено как набор модулей на едином стартовом экране:

| Модуль | Что делает | Бизнес-задача |
|--------|-----------|---------------|
| **Монтаж (Editor)** | Автосборка клипа под ритм трека (beat-detection), 9 эффектов, 10 фильтров, экспорт под площадки | Контент в ритм трека без ручного монтажа |
| **Уникализатор (VUB)** | Массовое создание N уникальных копий видео (9 вкладок настроек, многопоток, GPU) | Обход антидублей, заливка, арбитраж |
| **Замена титров (Cleaner)** | Детект и перекрытие чужих титров/логотипов + свои субтитры | Рерайт чужого контента |
| **Озвучка (TTS)** | Синтез речи нейроголосами (15 языков, Edge TTS) | Закадровый голос |
| **Дубляж** | Распознавание → перевод → переозвучка → синхронизация | Локализация роликов |
| **Шаблоны** | Готовые видео/фото-шаблоны и трендовые монтажи в один клик | Быстрый контент по образцу |
| **Pulsar Pro / Студия** | Профессиональный редактор монтажа + встроенный графический редактор (уровня Photoshop) | Ручная доводка |

## 3. Технические достижения

**Инфраструктура автоматизации:**
- Единая **очередь задач** с фоновым рендером — задачи не теряются при переключении модулей.
- **История** результатов (до 200 записей со снимком настроек, кнопка «открыть папку»).
- **GPU-ускорение** с автодетектом энкодера (NVIDIA NVENC / Intel QSV / AMD AMF) и fallback на CPU.
- **Мастер первого запуска** — установка Python и движков (Edge TTS, переводчик, yt-dlp) в один клик.
- Встроенный **FFmpeg/FFprobe** — приложение работает без внешних установок.

**Нейро- и умные функции:**
- Beat-detection через Python/librosa с fallback на равномерную нарезку.
- Распознавание речи (AssemblyAI/Whisper), офлайн-детект титров и логотипов (OpenCV/EAST).
- Умная рандомизация уникализации — гарантированное различие N копий, а не линейный прогон.
- Загрузка видео по ссылке (yt-dlp: TikTok / YouTube / Instagram и сотни площадок).

**Продакшен-качество:**
- Пресеты площадок (TikTok / Reels / Shorts / YouTube / Instagram) — формат, качество, лимит длины.
- CI-релизы через GitHub Releases + автообновление (electron-updater).
- Шифрование секретов (safeStorage/DPAPI), персистентные настройки и история.

## 4. Распределение работ по направлениям

Основные направления по объёму коммитов (всего 348):

| Направление | Коммитов* |
|-------------|-----------|
| Pulsar Pro (профредактор) | 38 |
| Студия (графредактор) | 36 |
| Уникализатор (VUB) + Титры | ~35 |
| Шаблоны / Тренды | ~20 |
| Замена титров (Cleaner) | 18 |
| Озвучка / Дубляж / Воронка | ~25 |
| Монтаж / Бит-детект | ~11 |
| Инфраструктура, фиксы, релизы | остальное |

\* сгруппировано по префиксам коммитов; ряд направлений пересекается.

**Динамика:** пик разработки пришёлся на недели W26–W27 (104 и 127 коммитов),
далее стабилизация и полировка (W28: 94, W29: 23) — типичный профиль «интенсивный
запуск → доводка до релиза».

## 5. Результат

- **Релизная версия 1.1.3**, собирается в установщик (NSIS) с автообновлением.
- Продукт **функционально целостен**: закрывает полный цикл производства видеоконтента
  офлайн, в одном окне, с пакетной обработкой и очередью.
- Реализовано **сверх исходного ТЗ**: озвучка, дубляж, загрузка по ссылке, библиотека
  шаблонов и трендов, встроенный графический редактор.

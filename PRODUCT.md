# Beatleap Atom (Pulsar) — описание продукта

Десктопное приложение для Windows (Electron + React + TypeScript) для создания и массовой
подготовки коротких видео под соцсети. Объединяет автомонтаж под музыку, уникализацию,
очистку чужих титров/логотипов, озвучку и дубляж — в одном окне, офлайн где возможно,
с собственным встроенным FFmpeg и опциональным Python для нейро-функций.

---

## 1. Из чего состоит приложение

Стартовый экран (**ModeSelector**) даёт выбор из пяти модулей и быстрый доступ к очереди,
истории и настройкам:

| Модуль | Назначение | Кому нужен |
|--------|-----------|-----------|
| **Монтаж (Editor)** | Автосборка клипа под ритм музыки + эффекты, фильтры, экспорт | Контент в ритм трека |
| **Уникализатор (VUB)** | Массовое создание N уникальных копий видео для обхода антидублей | Заливщики, арбитраж |
| **Замена титров (Cleaner)** | Детект и перекрытие чужих титров/логотипов + свои субтитры | Рерайт чужого контента |
| **Озвучка (TTS)** | Синтез речи из текста нейроголосами, наложение на видео | Закадровый голос |
| **Дубляж** | Перевод и переозвучка видео на другой язык с синхронизацией | Локализация роликов |

Сквозные элементы: **Очередь задач** (прогресс всех модулей, не теряется при переключении),
**История** (последние результаты с кнопкой «Открыть папку»), **Настройки**, **мастер первого
запуска** (установка Python и движков).

## 2. Монтаж (Editor) — автомонтаж под музыку

Главный модуль из исходного ТЗ: «видеоредактор с авто-синхронизацией по ритму».

### Сценарий
1. **Старт** → выбор видео (1–30 файлов, drag&drop, MP4/MOV/AVI, минимум ~5 сек суммарно),
   расстановка порядка клипов (перетаскивание, «Перемешать» / «По порядку»).
2. **Музыка** — встроенная библиотека (20+ треков по категориям POP/RAP/CINEMATIC/SPORT/…,
   с превью, BPM, обложкой) или свой MP3/WAV.
3. **Обработка** — анализ аудио (beat detection), нарезка клипов по битам, расстановка эффектов.
4. **Редактор** — правки в реальном времени.
5. **Экспорт** — выбор площадки, качества, папки, опционально уникализация копий.

### Анализ ритма
Аудио прогоняется через Python (librosa) → `beat_times`, `onset_times`, `tempo`.
При недоступности Python — fallback на равномерную нарезку (0.5 сек). Частота склеек зависит
от настроения (**Mood**): `mellow` (каждый 4-й бит), `natural` (каждый 2-й), `energetic`
(все биты + часть онсетов).

### Вкладка TOOLS
- **Videos** — добавить/удалить/переупорядочить исходники.
- **Tweak** — точная замена отдельного фрагмента (выбор диапазона по waveform, замена источника).
- **Duration** — длительность ролика (10/15/30 сек, 1/1.5/3 мин, весь трек).
- **Segment** — выбор сегмента трека (перетаскиваемое окно по waveform).
- **Mood** — частота склеек (mellow/natural/energetic).
- **Fade** — затемнение на старте/конце (none/in/out/all).
- **Format** — соотношение сторон 9:16 / 1:1 / 16:9.

### Вкладка EDIT — 9 эффектов
Сетка 3×3, у каждого эффекта уровень 0→1→2 (выкл/слабый/сильный) и мини-настройки
(интенсивность 0–100 % + вариант):

| Эффект | Варианты |
|--------|----------|
| Fast Cut | Резкие кадры / Строб |
| Flash | Белая / Чёрная вспышка |
| Zoom | Наезд / Отъезд / Пунш |
| Prism | RGB-смещение (резкое) |
| RGB | RGB-смещение (мягкое) |
| Boomerang | Реверс + сдвиг оттенка |
| Split | 2×2 / Зеркало / 2 полосы |
| Hue | Цветовая ротация |
| Speed | Разгон / Замедление / Постоянная |

Эффекты расставляются по битам (уровень 1 → ~20 % меток, уровень 2 → ~40 %, не ближе 0.5 сек).
Кнопка «Перемешать» переслучайнивает раскладку. Часть эффектов (Speed, Boomerang, Split,
FastCut) — превью-режим раскладки/тайминга.

### Вкладка FILTERS — 10 фильтров
Нет, Warm, Cool, Vintage, B&W, VCR, Glitch, Film, Light Leak, Vignette — с живым превью на
карточках и ползунком интенсивности 0–100 %.

### Экспорт
- **Пресеты площадок** (один клик задаёт формат/качество/лимит длины):
  TikTok 9:16 ≤3 мин, Reels 9:16 ≤90 сек, Shorts 9:16 ≤60 сек, YouTube 16:9 до 4K, Instagram 1:1 ≤60 сек.
- **Качество**: 720p / 1080p / 4K (размер зависит от формата).
- Выходной формат — MP4 (H.264 + AAC).
- Опциональная **уникализация** прямо при экспорте: создание N копий с лёгкими вариациями
  (цветосдвиг, шум, скорость, обрезка краёв, сдвиг аудио, зеркало, «видимая вариация»).

### Горячие клавиши
`Ctrl+S` экспорт · `Ctrl+Z` отмена · `Ctrl+N` новый проект · `Ctrl+O` добавить видео ·
`Ctrl+R` перемешать · `Space` play/pause · `1/2/3` вкладки.

## 3. Уникализатор (VUB) — массовая уникализация

Пакетная обработка: каждое видео × N вариаций = N выходных файлов со случайными, но
контролируемыми трансформациями. 9 вкладок настройки, многопоточная обработка, GPU-ускорение.

### Вкладка 1 — Загруженные видео
- Добавление: drag&drop, файловый диалог (MP4/MOV/AVI).
- **Загрузка по ссылке** *(добавлено в процессе)* — поле URL + кнопка «Скачать»: тянет видео
  с TikTok / YouTube / Instagram / Facebook и сотен других площадок через **yt-dlp**
  (склейка дорожек встроенным FFmpeg). Прогресс в процентах, автоустановка yt-dlp при первом
  использовании. Файл сохраняется в **`Загрузки/Beatleap/<метка времени>/`** и сразу попадает
  в список.
- У каждого видео — превью, имя, кнопка **«📁 Показать в папке»** *(добавлено в процессе)*
  (открывает проводник с выделенным файлом) и удаление.

### Вкладка 2 — Параметры видео (диапазоны рандомизации)
Двойные слайдеры min…max, каждый параметр включается отдельно:

| Параметр | Диапазон |
|----------|----------|
| Яркость | −50…+50 % |
| Контраст | −50…+50 % |
| Резкость | −50…+50 % |
| Громкость | −100…+100 % |
| Длительность (темп) | −50…+50 % |
| Поворот | −5…+5° (с зумом, без чёрных углов) |

Умная рандомизация: диапазон делится на N слотов по числу вариаций, слоты перемешиваются и
получают джиттер — чтобы N роликов гарантированно отличались, а не шли по возрастанию.

### Вкладка 3 — Видеоэффекты
- **Затемнение** (fade-in 1–30 сек, опц. фейд аудио).
- **Зеркало** (always / random по чётности вариации / never).
- **Полупрозрачная сетка** (прозрачность 1–50 %, цвет из палитры, ячейка 4–256 px).

### Вкладка 4 — Водяной знак
Загрузка PNG/GIF/MP4 + визуальный редактор зон: рисуешь прямоугольники-«коридоры», знак
ставится случайно в одной из зон.

### Вкладка 5 — Текст
**Spintax** `{вариант1|вариант2}` (с вложенностью) → у каждой вариации своя комбинация.
Шрифт, размер 8–200, цвет, позиция (верх/центр/низ).

### Вкладка 6 — Титры (авто-субтитры)
Распознавание речи через **AssemblyAI** (API-ключ в настройках, кэш на исходное видео),
языки ru/en/es/de/auto. Стиль: встроенные кириллические шрифты (Montserrat, Oswald, Rubik,
Fira Sans, Russo One) + системные, размер 24–140, цвет/подсветка, обводка 0–10, позиция X/Y,
**караоке** (пословная подсветка), CAPS, жирный, слов в строке 1–8, **подложка** (цвет,
прозрачность, ширина/высота/скругление). Пресеты: TikTok / Минимал / Жёлтый акцент / С обводкой
+ сохранение своих. Живое превью с перетаскиванием позиции.

### Вкладка 7 — Шаблон (склейка)
Вставка случайных клипов из выбранной папки каждые N секунд — разбивает монотонность.

### Вкладка 8 — Метаданные
Очистка отпечатков файла (copyright, даты, handler) и подстановка случайных Title/Artist/
Comment/Encoder — чтобы вариации не сопоставлялись как дубликаты.

### Вкладка 9 — Производительность и сохранение
Вариаций на видео 1–100; потоков 1…(ядра/2); GPU-режим auto/gpu/cpu (NVIDIA/Intel/AMD);
шаблон имени файла; папка вывода; запуск/отмена; таблица прогресса (имя / статус / %).

## 4. Замена титров (Cleaner)

Удаление чужих титров и логотипов из пакета видео, опционально — со своими субтитрами.

### Этапы
1. **Видео** — пакет файлов (drag&drop).
2. **Что искать** — титры/субтитры (с опцией «только меняющийся текст», чтобы не трогать
   статичные надписи) и/или логотипы.
3. **Способ перекрытия**:
   - **Delogo** — умное закрашивание/восстановление фона (качественно, медленнее).
   - **Блюр** — размытие зоны (сила 4–60).
   - **Плашка** — заливка прямоугольником (цвет, скругление 0–80).
4. **Свои титры** *(опц.)* — авто-субтитры поверх, со стилем из вкладки «Уникализатор → Титры»;
   размещение по найденной зоне (largest / lowest / highest) или по фикс-позиции.
5. **Зоны** — авто-детект на каждом видео **или** ручная разметка одних зон на весь пакет;
   порог уверенности 0–90 % (по умолч. 25).
6. **Сохранение** — папка вывода, «Анализировать и перекрыть», прогресс по файлам.

### Детектор (офлайн, Python + OpenCV)
- **Текст** — нейросеть EAST (любой угол), объединение боксов в строки.
- **Логотипы/статика** — анализ временной дисперсии кадров (низкая дисперсия = неподвижный
  оверлей). Работает без интернета.

## 5. Озвучка (TTS)

Синтез речи нейроголосами **Microsoft Edge TTS** (бесплатно, без ключа, нужен интернет).

- **15 языков**: ru, en, uk, es, de, fr, it, pt, pl, tr, ar, hi, ja, ko, zh — по 2–14 голосов
  (напр. ru: Светлана/Дария/Дмитрий; en: Aria, Jenny, Guy, Sonia, Ryan и др.).
- **Скорость** 0.5×–1.5×, кнопка **«Прослушать пример»**.
- **Наложение на видео** *(опц.)*: полная замена звука или микс с приглушением оригинала
  до 15 % («закадровый перевод»).
- Результат — MP3 (или MP4 при наложении), запись в историю.

## 6. Дубляж

Полная переозвучка видео на другом языке. Цепочка:
**ASR → сегментация → перевод → синтез → синхронизация по таймингам → (опц.) субтитры → склейка.**

1. **Распознавание** (AssemblyAI, ключ обязателен): аудио 16 кГц моно → слова с таймингами;
   язык оригинала auto или из списка (ru/en/es/de/fr/uk).
2. **Сегментация** — слова в фразы по знакам конца предложения или паузам (>6 сек).
3. **Перевод** — батчем через `deep-translator` (Google, бесплатно). Целевые языки: ru, en, uk,
   es, de, fr, it, pt, pl, tr, ja, zh.
4. **Синтез** — Edge TTS выбранным голосом целевого языка по сегментам.
5. **Синхронизация** — если озвучка длиннее слота (>5 %), ускорение через `atempo`; позиция
   через `adelay`; всё сводится `amix`. Опция «сохранить оригинал» — подмешивает исходный звук
   на 12 %.
6. **Субтитры** *(опц.)* — «выжигаются» в кадр (ASS, Montserrat, контур), видны в любом плеере.
7. **Склейка** — H.264 + AAC 192k, `+faststart`, файл `*_dub_<язык>.mp4`.

Прогресс по этапам (распознавание 5 % → перевод 25 % → озвучка 30–85 % → склейка 90 % → готово).

## 7. Инфраструктура

### Первый запуск (FirstRunSetup)
Мастер проверяет Python и ставит движки в один клик (с живым логом pip и прогрессом):
- **Edge TTS** (`edge-tts`) — озвучка/дубляж;
- **Перевод** (`deep-translator`) — дубляж/субтитры;
- **Загрузка по ссылке** (`yt-dlp`) — скачивание видео *(добавлено в процессе)*.

На Windows Python ставится автоматически через `winget` (нужен перезапуск); иначе — ссылка на
python.org. Мастер можно свернуть или вернуться к нему через Настройки.

### Бинарники и зависимости
- **FFmpeg + FFprobe** встроены (ffmpeg-static), распаковываются из asar — работают без установки.
- **Python** нужен только для нейро-функций (beat detection, TTS, перевод, детект оверлеев,
  загрузка по ссылке). Базовый монтаж и уникализация работают и без него (с fallback).

### Хранение
- **Секреты** (`vub-secrets.json`) — API-ключ AssemblyAI, шифруется через safeStorage/DPAPI.
- **Настройки** (`settings.json`) — GPU-режим, папки, язык, флаг первого запуска.
- **История** (`history.json`) — до 200 записей (тип, название, дата, папка, файлы, снимок настроек).

### Архитектура
Renderer (React/Zustand) ↔ IPC ↔ Main (Electron/Node: FFmpeg, файлы, spawn) ↔ Python-скрипты
(`tts.py`, `translate.py`, `beat_detect.py`, `detect_overlays.py`), выводящие JSON.

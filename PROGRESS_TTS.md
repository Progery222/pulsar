# PROGRESS — Модуль «Озвучка / Дубляж»

Новый раздел Pulsar: генерация речи (TTS) и дубляж видео на другой язык.
Движок синтеза подключаемый (XTTS-v2 по умолчанию: рус+многоязычность+клонирование;
альтернативы — Silero/Kokoro). Меняется в одном месте `python/tts.py`.

## Фаза 1: Озвучка из текста (TTS)
- [x] 1.1 Python-воркер `python/tts.py` (engine-абстракция xtts/silero/kokoro, JSON I/O, текст→wav)
- [x] 1.2 IPC `electron/ipc/tts.ts`: tts:synth (+ опц. наложение на видео через ffmpeg, GPU-энкодер)
- [x] 1.3 preload + регистрация в main
- [x] 1.4 UI-раздел `src/tts/TtsApp.tsx` (текст, язык, движок, скорость, клонирование, папка, наложить на видео)
- [x] 1.5 Карточка на старте + appMode 'tts' + интеграция очередь/история

## Фаза 2: Дубляж видео (ASR → перевод → TTS → склейка)
- [ ] 2.1 Перевод текста (offline argostranslate или API) — модуль `python/translate.py`
- [ ] 2.2 Пайплайн дубляжа: распознать речь (AssemblyAI) → перевести → озвучить → подложить дорожку
- [ ] 2.3 UI-вкладка «Дубляж»: исходный/целевой язык, сохранять субтитры
- [ ] 2.4 Тайминг: подгонка длительности фраз под исходные сегменты

## Зависимости (установка пользователем)
- XTTS-v2: `pip install coqui-tts` (+ torch). Первая генерация скачает модель (~1.8 ГБ).
- Альтернатива рус: `pip install silero` либо torch.hub silero.
- Перевод (Фаза 2): `pip install argostranslate` + языковые пакеты.
